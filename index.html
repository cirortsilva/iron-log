PASSO 1 — Criar projeto Supabase (gratuito)

Acesse supabase.com, crie uma conta gratuita e um novo projeto. Aguarde inicializar (~1 min).

PASSO 2 — Criar a tabela

No painel, vá em SQL Editor → New Query e execute:

CREATE TABLE sessions ( id TEXT PRIMARY KEY, date TEXT NOT NULL, treino TEXT NOT NULL, exercises JSONB NOT NULL, created_at TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE sessions ENABLE ROW LEVEL SECURITY; CREATE POLICY "allow_all" ON sessions FOR ALL USING (true) WITH CHECK (true);
PASSO 3 — Inserir as credenciais

Vá em Project Settings → API. Copie a Project URL e a chave anon public.
